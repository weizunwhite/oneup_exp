# oneup_exp

这是从智能背包/小车项目提炼出来的**可复用模板 + 核心库**，
用于后续项目快速复用或作为 AI 上下文仓库读取。

## 仓库结构

```
oneup_exp/
├─ lib/oneup_core/        # 核心库（UWB/跟随/电机/蜂鸣器/灯带/示教等）
├─ template/              # 可直接编译的模板工程
└─ README.md              # 使用说明
```

## 核心库包含

- UWB 解析与定位（距离/角度计算、偏置与反向支持）
- 跟随控制（滤波、迟滞、距离门限、指令保持）
- 电机 PWM 驱动（L298N，支持按模式切换速度）
- 蜂鸣器（非阻塞节奏与告警提示）
- WS2812 灯带（开关/颜色/亮度）
- 路径示教与回放
- 按钮事件（单击/双击/长按）
- OLED 显示、IMU 姿态、称重模块

## 快速开始

1) 进入模板工程

```
cd template
```

2) 修改 `include/oneup_config.h` 中的引脚与阈值
3) 使用 PlatformIO 编译并上传

```
pio run -t upload -e esp32dev
```

## 使用建议

- 新项目直接复制 `lib/oneup_core`
- 或以 `template/` 作为工程起点
